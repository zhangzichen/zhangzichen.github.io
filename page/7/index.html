应用后台处理

发表于 2015-03-02   |   分类于 技术篇

一个应用程序有其生命周期，周期中有着不同的状态，共分为以下几种：

未运行
–应用程序位于刚刚重启的设备商，或者在后台被系统清理掉，或者是运行时崩溃的都是未运行状态

活动
– 当前正在运行的程序状态为活动状态

后台
– 刚按下Home键的前期为后台，可以有5S的时间处理数据（可以申请延长）

暂停
– 处于后台的程序一会就会转为暂停

不活动
– 应用程序在两个状态之间临时过渡处于不活动状态

使用Safari或者电话服务

发表于 2015-02-14   |   分类于 技术篇 
// safari的使用
// 准备safari的URL
NSURL *safariUrl = [NSURL URLWithString:[NSString stringWithFormat:@"http://www.baidu.com"]];
// 判断safariUrl是否可以打开
if ([[UIApplication sharedApplication] canOpenURL:safariUrl]) {
// 打开系统自带的safariUrl
[[UIApplication sharedApplication] openURL:safariUrl];
} else {
NSLog(@"无法打开safari");
}


// 电话的使用
// 准备打开拨号程序的url
NSURL *telUrl = [NSURL URLWithString:@"tel://555-1234"];
// 判断可否打开电话程序
if ([[UIApplication sharedApplication] canOpenURL:telUrl]) {
// 打开url
[[UIApplication sharedApplication] openURL:telUrl];
} else {
NSLog(@"无法打开拨号");
}
