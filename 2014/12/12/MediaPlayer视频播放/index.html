MediaPlayer视频播放

发表于 2014-12-12   |   分类于 技术篇
  1. 视频播放
  2. 需要引入的框架和头文件：#import
  3. 播放视频的时候，我们需要使用MPMoviePlayerViewController来控制
  4. 虽然我们需要使用MPMoviePlayerViewController来播放，但是播放视频的一些属性需要在moviePlayer属性中设置
  5. MPMoviePlayerViewController的初始化方法:

- (id)initWithContentURL:(NSURL *)contentURL; // 和播放音乐一样，都是使用url，但是不同的是音乐可以使用网络数据初始化，而视频不可以

@property(nonatomic, readonly) MPMoviePlayerController *moviePlayer; // 只有这一个属性，并且还是只读的，所以我们需要通过设置这个属性的一些属性来设置视频播放器

MPMoviePlayerController的一些设置：

- (id)initWithContentURL:(NSURL *)url;

@property(nonatomic, copy) NSURL *contentURL; // 视频的url地址
// The view in which the media and playback controls are displayed.
@property(nonatomic, readonly) UIView *view; // 通过设置view的一些属性来设置视频播放的一些显示
// A view for customization which is always displayed behind movie content.
@property(nonatomic, readonly) UIView *backgroundView; // 通过设置backgroundView的一些属性来设置视频播放的背景
// Returns the current playback state of the movie player.
@property(nonatomic, readonly) MPMoviePlaybackState playbackState; // 获取播放状态
// Returns the network load state of the movie player.
@property(nonatomic, readonly) MPMovieLoadState loadState; // 获取加载时的状态
// The style of the playback controls. Defaults to MPMovieControlStyleDefault.
@property(nonatomic) MPMovieControlStyle controlStyle; // 设置视频播放页面的样式
// 1112. Defaults to MPMovieRepeatModeNone.
@property(nonatomic) MPMovieRepeatMode repeatMode; // 设置是否循环播放
// Indicates if a movie should automatically start playback when it is likely to finish uninterrupted based on e.g. network conditions. Defaults to YES.
@property(nonatomic) BOOL shouldAutoplay; // 进入视频自动播放
// Determines if the movie is presented in the entire screen (obscuring all other application content). Default is NO.

// Determines how the content scales to fit the view. Defaults to MPMovieScalingModeAspectFit.
@property(nonatomic) MPMovieScalingMode scalingMode; // 设置缩放模式


// The types of media in the movie, or MPMovieMediaTypeNone if not known.
@property(nonatomic, readonly) MPMovieMediaTypeMask movieMediaTypes; // 播放是是音频还是视频

// The duration of the movie, or 0.0 if not known.
@property(nonatomic, readonly) NSTimeInterval duration; // 视频的播放时长

// The currently playable duration of the movie, for progressively downloaded network content.
@property(nonatomic, readonly) NSTimeInterval playableDuration; // 当前的播放时间

// The start time of movie playback. Defaults to NaN, indicating the natural start time of the movie.
@property(nonatomic) NSTimeInterval initialPlaybackTime; // 开始播放时间
// The end time of movie playback. Defaults to NaN, which indicates natural end time of the movie.
@property(nonatomic) NSTimeInterval endPlaybackTime; // 结束播放时间



// Setting this property to YES before the movie player's view is visible will have no effect.
@property(nonatomic, getter=isFullscreen) BOOL fullscreen;
- (void)setFullscreen:(BOOL)fullscreen animated:(BOOL)animated;  // 设置是否全屏，是否使用动画

#pragma mark - 实现---显示/隐藏状态栏和导航栏
- (void)showHideBar {
// 代码顺序不可错，否则会出问题
BOOL showFlag = self.navigationController.navigationBar.hidden ? NO : YES;
// 状态栏
[[UIApplication sharedApplication] setStatusBarHidden:showFlag withAnimation:UIStatusBarAnimationSlide];
// 导航栏
[self.navigationController setNavigationBarHidden:showFlag animated:YES];
}
